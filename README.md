## À propos

**Anarchic** est un outil OSINT puissant pour la recherche et l'exploitation de bases de données. Conçu pour les enquêteurs numériques, les pentesteurs et les professionnels de la cybersécurité.

### Fonctionnalités prévues

| Module | Description |
|--------|-------------|
| 🔍 **DB Search** | Recherche multi-sources dans les bases de données exposées |
| 👤 **Identity** | Cross-referencing d'identités et de comptes |
| 🌐 **Social** | Analyse de profils et traces sur les réseaux sociaux |
| 📡 **Network** | Scan réseaux, WHOIS, DNS, certificats |
| 🔑 **Leaks** | Vérification de fuites de données et credentials |
| 📄 **Export** | Export des résultats (CSV, JSON, HTML) |

### Stack

- **Langage :** Python
- **Plateformes :** Windows (.exe) / Linux
- **Base :** SQLite embarqué + APIs distantes
- **Interface :** CLI + (future GUI)
